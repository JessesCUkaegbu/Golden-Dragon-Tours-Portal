# Golden Dragon Tours Portal

A mini online application and ticket booking portal built with **Django, HTML, CSS, Bootstrap, and JavaScript**.

This project is designed for a travel agency workflow where users can scan a QR code, access the portal, create an account or log in, fill out a ticket form, and receive a digital ticket with a barcode after submission.

---

## Features

- User registration and login
- QR-code-based access to the portal
- Ticket application form
- Ticket generation after submission
- Barcode display for each submitted ticket
- User dashboard to manage ticket activity
- Clean modern UI using Bootstrap and custom styling

---

## Tech Stack

- **Backend:** Django
- **Frontend:** HTML, CSS, Bootstrap, JavaScript
- **Database:** SQLite
- **Barcode Generation:** `python-barcode`, `Pillow`

---

## Project Workflow

1. User scans the travel agency QR code
2. User is redirected to the application portal
3. User creates an account or logs in
4. User clicks the **Get Ticket** button
5. User fills out and submits the ticket form
6. A success popup displays the ticket details and barcode
